# 📁 05_Search_Sort_Filter

## 📖 Overview

Welcome to the **Search, Sort & Filter** module! 🔍📊

This section focuses on organizing, searching, and filtering data using NumPy. These operations are essential for data preprocessing, cleaning, and analysis, making it easier to extract meaningful information from datasets. Mastering these concepts is a key step toward real-world data science and machine learning applications.

---

## 🎯 Objectives

By completing this module, you will learn how to:

- Sort arrays in different ways.
- Search for specific elements and their positions.
- Find maximum and minimum values efficiently.
- Filter data based on given conditions.
- Remove duplicate values from datasets.
- Count the frequency of elements.
- Extract unique values for data analysis.

### 🔹 Sorting Arrays

## 🛠️ Functions Practiced

| Function | Purpose |
|----------|---------|
| `np.sort()` | Sort arrays in ascending order |
| `np.argsort()` | Return indices that would sort an array |
| `np.argmax()` | Find the index of the maximum value |
| `np.argmin()` | Find the index of the minimum value |
| `np.where()` | Locate elements matching a condition |
| `np.unique()` | Find unique elements and remove duplicates |
| `np.unique(return_counts=True)` | Count the frequency of unique elements |
| Boolean Masking | Filter data based on conditions |

# 📁 11_Image_Processing

## 📖 Overview

Welcome to the **Image Processing with NumPy** module! 🖼️

This section introduces the fundamentals of digital image processing using NumPy. Since an image is essentially a matrix of pixel values, NumPy provides powerful tools for manipulating images efficiently. In this module, you'll perform common image transformations, filtering operations, and basic computer vision tasks using only NumPy.

---

## 🎯 Objectives

By completing this module, you will learn how to:

- Represent images as NumPy arrays.
- Perform pixel-level image manipulations.
- Apply basic image enhancement techniques.
- Use array slicing for image editing.
- Understand simple image filtering concepts.
- Build creative image-based mini projects using NumPy.

## 🛠️ Functions Practiced

| Function | Purpose |
|----------|---------|
| `np.array()` | Create image matrices |
| `np.zeros()` | Generate blank images |
| `np.ones()` | Create white image matrices |
| `np.clip()` | Limit pixel values within a valid range |
| `np.mean()` | Compute grayscale intensity and blur calculations |
| `np.flip()` | Flip images horizontally or vertically |
| `np.transpose()` | Rearrange image dimensions |
| `np.pad()` | Add padding for filtering operations |
| Array Slicing | Crop and manipulate image regions |

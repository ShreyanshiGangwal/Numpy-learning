# 📁 09_Linear_Algebra

## 📖 Overview

Welcome to the **Linear Algebra** module! 📐🔢

This section introduces the mathematical foundations of NumPy used in Data Science, Machine Learning, Computer Vision, Artificial Intelligence, and scientific computing. You'll learn how to perform matrix operations, solve systems of equations, and analyze matrices using powerful linear algebra functions provided by NumPy.

---

## 🎯 Objectives

By completing this module, you will learn how to:

- Perform matrix multiplication and vector operations.
- Compute dot and cross products.
- Transpose matrices.
- Calculate determinants and inverses.
- Solve systems of linear equations.
- Find eigenvalues and eigenvectors.
- Determine the rank of a matrix.
- Apply linear algebra concepts to build a simple recommendation score matrix.

## 🛠️ Functions Practiced

| Function | Purpose |
|----------|---------|
| `np.matmul()` / `@` | Matrix multiplication |
| `np.dot()` | Compute dot product |
| `np.cross()` | Compute cross product |
| `np.transpose()` / `.T` | Transpose a matrix |
| `np.linalg.det()` | Calculate determinant |
| `np.linalg.inv()` | Find matrix inverse |
| `np.linalg.solve()` | Solve system of linear equations |
| `np.linalg.eig()` | Compute eigenvalues and eigenvectors |
| `np.linalg.matrix_rank()` | Determine matrix rank |

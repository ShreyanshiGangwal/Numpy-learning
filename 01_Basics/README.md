# 📁 01_Basics

## 📖 Overview

Welcome to the **NumPy Basics** module! 🚀

This section covers the fundamental concepts of NumPy, including array creation, array properties, data types, and built-in functions used to initialize arrays. It serves as the foundation for all advanced NumPy operations and helps build a strong understanding of numerical computing with Python.

---

## 🎯 Objectives

By completing this module, you will learn how to:

- Create NumPy arrays from Python lists.
- Work with 1D, 2D, and 3D arrays.
- Understand important array properties.
- Generate arrays using different NumPy functions.
- Convert between Python lists and NumPy arrays.
- Create arrays with specific data types.
- Compare the efficiency of Python lists and NumPy arrays.

---

### 🔹 Array Creation
- Create NumPy array from a Python list
- Create 1D array

## 🛠️ Functions Practiced

| Function | Purpose |
|----------|---------|
| `np.array()` | Create NumPy arrays |
| `np.zeros()` | Create an array filled with zeros |
| `np.ones()` | Create an array filled with ones |
| `np.full()` | Create an array filled with a specific value |
| `np.arange()` | Generate evenly spaced values within a range |
| `np.linspace()` | Generate evenly spaced values between two limits |
| `np.eye()` | Create an identity matrix |
| `tolist()` | Convert NumPy array to Python list |

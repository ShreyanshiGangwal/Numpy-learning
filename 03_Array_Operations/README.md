# 📁 03_Array_Operations

## 📖 Overview

Welcome to the **Array Operations** module! 🚀

This section focuses on performing mathematical and statistical operations on NumPy arrays. You'll learn how to manipulate array elements, apply arithmetic operations, calculate statistical measures, compare arrays, and normalize data. These operations are fundamental for data analysis, machine learning, and scientific computing.

---

## 🎯 Objectives

By completing this module, you will learn how to:

- Perform arithmetic operations on NumPy arrays.
- Apply mathematical transformations to array elements.
- Calculate statistical values from arrays.
- Work with scalar operations.
- Compare multiple arrays.
- Compute cumulative calculations.
- Normalize data for analysis and machine learning.

---

## 📚 Topics Covered

### 🔹 Arithmetic Operations
- Array Addition
- Array Subtraction
- Array Multiplication
- Array Division

### 🔹 Mathematical Operations
- Square every element
- Cube every element

### 🔹 Statistical Operations
- Maximum Value (`max()`)
- Minimum Value (`min()`)
- Mean (`mean()`)
- Median (`median()`)
- Sum (`sum()`)

### 🔹 Scalar Operations
- Add a scalar value to an array
- Multiply a matrix by a scalar

### 🔹 Array Comparison
- Compare two arrays element-wise

### 🔹 Cumulative Operations
- Cumulative Sum (`cumsum()`)
- Cumulative Product (`cumprod()`)

### 🔹 Data Normalization
- Normalize array values using mathematical formulas

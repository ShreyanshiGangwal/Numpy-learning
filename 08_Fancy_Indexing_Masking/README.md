# 📁 08_Fancy_Indexing_Masking

## 📖 Overview

Welcome to the **Fancy Indexing & Masking** module! 🎯🧩

This section introduces powerful NumPy techniques for selecting, extracting, and modifying data efficiently. Fancy indexing and Boolean masking allow you to work with specific rows, columns, or elements based on indices and conditions, making them essential tools for data cleaning, preprocessing, and analysis.

---

## 🎯 Objectives

By completing this module, you will learn how to:

- Select elements using index arrays.
- Extract multiple rows and columns efficiently.
- Modify array values using Boolean masking.
- Filter data based on different conditions.
- Handle missing (`NaN`) values.
- Detect unusual values (outliers).
- Build practical applications such as a grading system using masks.

---

## 📚 Topics Covered

### 🔹 Fancy Indexing
- Select elements using index arrays
- Extract multiple rows
- Extract multiple columns

### 🔹 Boolean Masking
- Replace values using masking
- Filter negative values
- Filter prime numbers
- Replace `NaN` values
- Highlight values greater than the mean
- Find outliers
- Create a grading system using masks

| Function | Purpose |

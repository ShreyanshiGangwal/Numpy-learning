## 🛠️ Functions Practiced

| Function | Purpose |
|----------|---------|
| `np.mean()` | Calculate the average value |
| `np.median()` | Find the middle value |
| `np.var()` | Calculate variance |
| `np.std()` | Calculate standard deviation |
| `np.percentile()` | Find percentiles |
| `np.corrcoef()` | Calculate correlation between variables |
| `np.histogram()` | Generate histogram data |
| `np.where()` | Detect anomalies and filter values |
| `np.abs()` | Compute absolute values |
| `np.sum()` | Summarize data |
| `np.min()` | Find minimum value |
| `np.max()` | Find maximum value |

> **Note:** NumPy does not provide a built-in `mode()` function. Mode can be calculated using techniques such as `np.unique(..., return_counts=True)`.

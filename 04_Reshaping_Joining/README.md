# 📁 04_Reshaping_Joining

## 📖 Overview

Welcome to the **Reshaping & Joining** module! 🚀

This section focuses on transforming the structure of NumPy arrays without changing their data. You will learn how to reshape arrays into different dimensions, flatten multidimensional arrays, combine multiple arrays, split arrays into smaller parts, and efficiently manipulate array layouts for data preprocessing and analysis.

---

## 🎯 Objectives

By completing this module, you will learn how to:

- Reshape arrays into different dimensions.
- Convert multidimensional arrays into one-dimensional arrays.
- Understand the difference between `ravel()` and `flatten()`.
- Join multiple arrays using different stacking techniques.
- Split arrays into equal and unequal parts.
- Repeat and tile array elements.
- Swap rows and columns of a matrix.

---

## 📚 Topics Covered

### 🔹 Reshaping Arrays
- Reshape 1D array to 2D
- Reshape 2D array to 3D
- Flatten a matrix into a 1D array

### 🔹 Flattening Methods
- `ravel()`
- `flatten()`

### 🔹 Joining Arrays
- Horizontal Stacking (`hstack()`)
- Vertical Stacking (`vstack()`)
- Concatenating Arrays (`concatenate()`)

### 🔹 Splitting Arrays
- Split arrays into equal parts
- Split arrays into unequal parts

### 🔹 Array Repetition
- Repeat array elements (`repeat()`)
- Tile arrays (`tile()`)

### 🔹 Matrix Manipulation
- Swap rows and columns
- Matrix Transpose

---

## 🛠️ Functions Practiced

| Function | Purpose |
|----------|---------|
| `reshape()` | Change the shape of an array |
| `ravel()` | Flatten an array (returns a view when possible) |
| `flatten()` | Flatten an array (returns a copy) |
| `np.hstack()` | Stack arrays horizontally |
| `np.vstack()` | Stack arrays vertically |
| `np.concatenate()` | Join arrays along an existing axis |
| `np.split()` | Split an array into equal parts |
| `np.array_split()` | Split an array into unequal parts |
| `np.repeat()` | Repeat array elements |
| `np.tile()` | Repeat the entire array multiple times |
| `transpose()` / `.T` | Swap rows and columns |

# 📁 02 - Indexing & Slicing

## 📖 Overview

The **Indexing & Slicing** module focuses on accessing, extracting, and modifying data within NumPy arrays. It introduces powerful techniques for retrieving specific elements, selecting rows and columns, manipulating subarrays, and performing efficient data transformations. Mastering indexing and slicing is essential for data analysis, machine learning, and scientific computing.

---

## 🎯 Objectives

By completing this module, you will learn how to:

- Access individual elements in NumPy arrays.
- Slice arrays to retrieve specific portions of data.
- Extract rows, columns, and submatrices.
- Modify array elements efficiently.
- Reverse arrays, rows, and columns.
- Work with diagonal and corner elements of matrices.
- Create patterns using slicing techniques.

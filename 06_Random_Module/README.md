# 📁 06_Random_Module

## 📖 Overview

Welcome to the **NumPy Random Module**! 🎲🎯

This module introduces NumPy's powerful random number generation capabilities. Random data is widely used in simulations, machine learning, data science, gaming, image processing, and statistical analysis. In this section, you'll learn how to generate random values, create random datasets, simulate real-world events, and build small practical applications using NumPy's random functions.

- Create random arrays and matrices.
- Generate random values within a specified range.

- Build simple projects using NumPy's random module.

## 🛠️ Functions Practiced

| Function | Purpose |
|----------|---------|
| `np.random.randint()` | Generate random integers within a specified range |
| `np.random.rand()` | Generate random floating-point numbers between 0 and 1 |
| `np.random.uniform()` | Generate random floating-point numbers within a custom range |
| `np.random.choice()` | Randomly select elements from an array |
| `np.random.shuffle()` | Shuffle the elements of an array in place |
| `np.random.seed()` | Set a seed for reproducible random results *(optional)* |
